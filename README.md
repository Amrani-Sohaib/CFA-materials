#  CFA Level I - Study Materials

Welcome to the **CFA Materials** repository. This repository is designed to help individuals prepare for the **CFA Level I** (Chartered Financial Analyst) exam by providing interactive Jupyter Notebooks covering the entire curriculum.

##  Available Topics

### Quantitative Methods
- [Quantitative Methods I](QuantitativeMethods_I.ipynb) — Time value of money, probability distributions, and foundational concepts.
- [Quantitative Methods II](QuantitativeMethods_II.ipynb) — Hypothesis testing, regression analysis, and financial modeling.

### Economics
- [Economics I](Economics_I.ipynb) — Demand and supply analysis, microeconomic and macroeconomic fundamentals.
- [Economics II](Economics_II.ipynb) — Monetary and fiscal policy, inflation control, and economic growth.

### Financial Reporting and Analysis
- [Financial Reporting and Analysis I](Financial_Reporting_and_Analysis_I.ipynb) — Financial statements and reporting standards.
- [Financial Reporting and Analysis II](Financial_Reporting_and_Analysis_II.ipynb) — Income statements, balance sheets, and cash flow analysis.
- [Financial Reporting and Analysis III](Financial_Reporting_and_Analysis_III.ipynb) — Inventories, long-lived assets, and taxes.
- [Financial Reporting and Analysis IV](Financial_Reporting_and_Analysis_IV.ipynb) — Financial analysis techniques and ratios.

### Corporate Finance
- [Corporate Finance I](Corporate_Finance_I.ipynb) — Capital budgeting, cost of capital, and leverage.
- [Corporate Finance II](Corporate_Finance_II.ipynb) — Working capital management and corporate governance.

### Portfolio Management
- [Portfolio Management I](Portfolio_Management_I.ipynb) — Portfolio theory, risk-return tradeoff, and diversification.
- [Portfolio Management II](Portfolio_Management_II.ipynb) — Asset allocation and portfolio construction.

### Equity Investments
- [Equity Investments I](Equity_Investments_I.ipynb) — Market organization, security market indices, and market efficiency.
- [Equity Investments II](Equity_Investments_II.ipynb) — Equity valuation concepts and techniques.

### Fixed Income
- [Fixed Income I](Fixed_Income_I.ipynb) — Bond features, pricing, and yield measures.
- [Fixed Income II](Fixed_Income_II.ipynb) — Interest rate risk, credit risk, and bond valuation.

### Derivatives
- [Derivatives](Derivatives.ipynb) — Forwards, futures, options, and swaps.

### Alternative Investments
- [Alternative Investments](Alternative_Investments.ipynb) — Real estate, private equity, hedge funds, and commodities.

### Ethics and Professional Standards
- [Ethical and Professional Standards](Ethical_and_Professional_Standards.ipynb) — CFA Institute Code of Ethics and Standards of Professional Conduct.

##  Getting Started

### Prerequisites

- Python 3.9+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/CFA-materials.git
cd CFA-materials

# Create a virtual environment
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
# .venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
```

### Run the notebooks

Open any notebook directly in **VS Code** or launch Jupyter:

```bash
jupyter notebook
```
